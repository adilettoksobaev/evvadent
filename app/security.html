3М-ESPE

Pentamix™ 2, производства компании 3M ESPE, представляет собой автоматическое смесительное устройство для смешивания стоматологических оттискных материалов Penta™.

После запуска устройства Pentamix™ специальное внутреннее устройство открывает пакет из фольги после подачи необходимого давления, устанавливаемого поршнем.

Коннекторы на уплотнительных колпачках PentaMatic предназначены для правильной установки смесительного наконечника.
